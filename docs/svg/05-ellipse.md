# Drawing Rectangles with SVG



```html
<svg height="200" width="200">
  <ellipse cx="100" cy="80" rx="100" ry="50"
           fill="cyan" stroke-width="3"; stroke="purple" />
</svg>
```

This example will generate the following line:

<svg height="200" width="200">
  <ellipse cx="100" cy="80" rx="90" ry="50"
           fill="cyan" stroke-width="3"; stroke="purple" />
</svg>
