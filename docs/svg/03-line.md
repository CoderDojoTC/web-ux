# Drawing Lines with SVG

```html
<svg height="100" width="100">
  <line x1="20" y1="10" x2="90" y2="80" stroke="green" stroke-width="5"/>
</svg>
```

This example will generate the following circle:

<svg height="100" width="100">
  <line x1="20" y1="10" x2="90" y2="80" stroke="green" stroke-width="5"/>
</svg>
